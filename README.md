# Digitaler Unterlagen-Ordner

PDF document manager for Windows: tree view, import, export, and table of contents generation.

A desktop application (Python + Tkinter) that organizes PDF documents in a structured tree,
supports merge/split/compress operations, and generates bookmark-based tables of contents.
