### FEATURES_REQUIRED.md — critical-path validation
All 7 paths are implemented and exercised by passing suites. **Integration caveats this pass surfaced:**
- **Path 1/2/5 (Import → selection → Export):** in **group-by-tag view**, shift-range selection captures synthetic `__tag__*` header ids and duplicate real ids; these feed *Set status (many)* and *Ctrl+E export*, sending non-existent ids to the backend (raw English `node not found: __tag__…`) and inflating the "{n} Elemente" count — a silent-failure class (**F-5**). Today's F2 fix limited range to *visible rows* but not to *real* rows in group view.
- **Path 3 (Preview & compression):** Ctrl+wheel zoom (documented gesture) is **dead for the whole session** (**H-1**); the proactive no-gain sweep runs **only once per window** so re-opening a second document in the same window leaves cheap undecided leaves red-dotted (**F-7**).

## Summary

**Findings by severity:** 0 Critical · **3 High** · **13 Medium** · **20 Low** = **36 total**.

### Top 3 High-priority fixes
1. **H-1 — Restore Ctrl+wheel zoom** (`App.jsx:75-85`): gate the wheel effect on `ready=!!state` (or callback ref). A *regression from today's commit `1f9f667`*; one-line fix + regression test. Highest user-visible impact.
2. **H-2 — Test the archive actual-read bomb cap** (`archives.py` ZIP/TAR): the primary liar-entry defence currently has 0 coverage; a refactor could silently disable it.
3. **H-3 — Test the invalid-archive `ValueError` contract** (`archives.py`): cheap, locks the translation callers depend on.

### Top 2 architectural improvements
1. **View-mode selection/nav coherence** (M-6 + M-7 + F-5 caveat): feed `treeForView` to `useKeyboard` and filter synthetic/duplicate ids out of group-view ranges, with an `ignoreCollapsed` flag mirroring `forceExpand`. Closes a silent-failure class where users act on documents they never saw selected.
2. **`useModal`/shortcut-gating discipline** (M-4 + M-5): ref the `onClose` so the focus-trap doesn't re-arm on every parent render, and route F2 through the same modal gate as the other shortcuts — makes the focus-management layer match the (already strong) logic layer.

### Quick wins (high impact / low effort)
- **H-1** wheel-effect dep (1 line) — user-visible regression.
- **L-1** `.bak` fail-open guard (a few lines) — data-integrity.
- **M-1** `.odt/.ods/.odp` advertised-but-rejected — add 3 exts to the COM allowlist *or* drop them from the dialog filters.
- **L-18 / L-19 / M-13** docs hygiene — package.json version, regenerate note, Project Directory table.
- **M-8** no-gain sweep dep `[session, state?.tree?.id]` (1 line).
