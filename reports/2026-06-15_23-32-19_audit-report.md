audit cycles and that holds: imports are content-sniffed before extensions are trusted,
every container path (zip/tar/eml/msg) carries member-count + decoded-byte bomb caps
with a *fail-closed* OOXML `.rels` external-target pre-scan, render and compression
paths clamp a per-page pixel budget against MediaBox allocation bombs, the React UI has

in app code.

web suite grew 216 → **281** tests. Independent re-review surfaced **one Low,

## What was verified clean (with evidence)

**Security**
- **Bomb guards intact** — zip/tar/email enforce member-count + declared-size pre-check *and* an actual-read running-total early abort (`archives.py:59-95` zip, `115-153` tar, `226-339` email incl. body bytes, refuse-before-convert). Dedicated `_ArchiveTooLarge` (`archives.py:38-40,146-147`) can't be swallowed by the per-member catch. Confirmed.
- **Zip-Slip / path traversal** — no archive member is written to a member-named path; tar uses in-memory `extractfile` + `os.path.basename` (`archives.py:128`); `import_bytes` uses `os.path.basename(name)` (`core/api.py:988`); `materialize_subset` sanitizes illegal chars + Windows-reserved names + trailing dots before forming a path (`core/api.py:885-892`). No `extractall`. Confirmed.
- **OOXML `.rels` external-target scan** — fail-closed for ZIPs (entry-cap, oversized-`.rels` truncation, any scan exception → `SCAN_UNREADABLE`); hyperlinks exempted; legacy-OLE/ODF fail-open deliberate; caller refuses before any COM open (`converters.py:169-234`). Confirmed.
- **Content-vs-extension gating** — `verify_content_matches_extension` runs on **both** the path branch (`importer.py:105-110`) and the bytes branch (`importer.py:129`); MZ/ELF/Mach-O/shebang refused (`importer.py:33-65`). Confirmed.
- **COM Office leak safety** — `Quit()` in `finally` on every path (`converters.py:268,285,299`); `out_path`/`tmp_dir` cleaned in `finally` (`converters.py:304-311`). Confirmed.
- **Pixel-budget clamps** — `capped_dpi` (`services/render.py:30-40`) on every preview path; the compression raster path now clamps height too (`compress_pdf_bytes.py:168-169`), closing the prior width-only gap. Confirmed.
- **No injection surface** — no `eval`/`exec`/`pickle`/`yaml.load`/`os.system`/`subprocess`/`shell=True` in source (only `subprocess` is in a test). Variants use a stored ZIP, "not pickle" (`variant_blobs.py:6`). Confirmed.
- **No web XSS sinks** — no `dangerouslySetInnerHTML`/`innerHTML`/`document.write`/`eval` in `webui/src`. Confirmed.
- **HTML→PDF link guard** — `block_remote_link` allows only `data:`/`cid:`, blocking remote + `file://` (LFI/SSRF) (`converters.py:98-114`). Confirmed.
- **PII out of logs** — email parsing logs only structure/lengths, never subject/from/body (`archives.py:198-199,219-220`). Confirmed.

**Correctness / concurrency**
- `_restore_from_bak` requires **both** `%PDF` header and `%%EOF` tail before accepting an interrupted save (`core/api.py:292-304`); `_save_through_lock` aborts rather than overwrite-without-backup on a failed prior read (`api.py:729-740`). Confirmed.
- `_after_mutate` revives cancel tokens on the failure path too (`api.py:448-460`); cross-session id-sharing guard prevents one window's delete from killing another's compression (`api.py:438-445`, mirrored in `close_session` `api.py:786-797`). Confirmed.
- `RenderCache` mutations iterate a materialized snapshot, not the live dict (`render_service.py:94,109`); pool/executor double-checked-locked init; generation token abandons stale fills mid-batch. Confirmed.
- `compress_pdf_bytes` returns the original on any page-count drop (`compress_pdf_bytes.py:59-72`); `txt_to_pdf` always advances (no infinite loop on an over-wide glyph) and re-sets the font after `showPage` (`converters.py:67-90`). Confirmed.
- `FileLock` shares only `FILE_SHARE_READ`, maps `ERROR_SHARING_VIOLATION` → `FileInUseError`, bounded partial-write loop (`file_lock.py:39-100`). Confirmed.

**Architecture / workspace conventions**
- No UI import in `core/`/`services/`/`formats/` — logic layer is UI-free per the CLAUDE.md contract; React business logic lives in `webui/src/lib/`. Confirmed.
- `manual_tests/` present (README + 01–08); `docs/data-model.html` updated 2026-06-15, carries the `has_source` row; `FEATURES_REQUIRED.md`, `.gitattributes` (LF), pinned `requirements.txt`, CLAUDE.md back-link + directory listing all present. No new convention gaps (the "manual_tests 01–04 describe legacy Tk flows" item is a known, documented limitation). Confirmed.
- No tracked build artifacts; `.gitignore` excludes `*.belegtool` (user PII) and `Beta-Feedback/`. Confirmed.
